# Contributing

## You need

- docker

## Tech Stack

- python
- docker
- Flask

## Running locally

1. 'sh cli.sh build-image'

2. 'sh cli.sh run-container'

## Deploy

1. Log on GCP with gcloud docker image 'gcloud auth login'

2. Set the GCP project 'gcloud config set project o-que-le-agora'

3. Run 'gcloud app deploy'
